    \begin{block}{Context}
	\textbf{Outdoor unknown off-road navigation stakes in mobile robotics applications are multiples:}

  \vspace{1cm}

  \begin{itemize}
    \setlength{\itemsep}{30pt}
    \item The need to specify \textbf{ground geometry}, its nature (e.g., slippery ground), as well as the presence and nature of \textbf{dynamic objects} to adjust the mapping models and potentially to avoid them during path generation.
    \item \textbf{Changes in the environment} (variations in luminosity, rain, fog, etc) do not always allow the continuous use of the same sensors. Therefore, sensors selection but also multi-sensor fusion, is essential.
    \item The \textbf{noisy nature of acquisition sensors} must be the subject of careful consideration to provide robust information (possibly guaranteed using Interval Analysis ~\cite{interval}) on which path generation can be based. The context of the described environment corresponds to applications such as construction or farming sites where vehicles evolve jointly with human operators, applications where safety is the top priority.
  \end{itemize}
  \end{block}

    \vspace{1cm}

    	\begin{block}{A robust special sensor: The micro-wave RADAR}

      \vspace{0.8em}
      \begin{center}
        \includegraphics[width=1\linewidth]{images/Generalshema}\\[4pt]
        {\small\itshape Figure 1. Overview of radar integration in SLAM.}
      \end{center}
		
    \vspace{0.5cm}

    Similar to LiDAR, RADAR is an electromagnetic-based active sensor, that returns the distance to an obstacle using the time of flight between signal transmission and reception. The difference is that the wave frequencies lie in the radio-frequency range, between 3 MHz and 300 GHz (76 - 81 GHz for automotive RADAR), whereas LiDAR wave frequency operates in the infrared range, at approximately 10,000 times higher frequencies. This difference in wave frequency translates into sensor property where RADAR:

    \vspace{1.5cm}

    \begin{columns}

    \begin{column}{0.65\linewidth}

    \begin{itemize}
      \setlength{\itemsep}{15pt}
      \item Is less sensitive to signal attenuation in the atmosphere, allowing \textbf{higher maximum range and greater robustness} to weather conditions such as rain or fog.
      \item Returns the \textbf{Doppler velocity} of obstacles.
      \item Can potentially \textbf{penetrate certain materials} depending on the type of RADAR.
      \item Is \textbf{cheaper} than LiDAR, making it attractive for industrial applications.
    \end{itemize}

    \vspace{1.5cm}

    However, RADAR also has some drawbacks when compared to LiDAR:

    \vspace{1.5cm}
    
    \begin{itemize}
      \setlength{\itemsep}{15pt}
      \item RADAR has a \textbf{lower detection accuracy} compared to LiDAR requiring more careful noise modelling.
      \item It has \textbf{low spatial resolution}, which makes it complicated to detect small detection for weak Radar Cross Section target (RCS) depending on shape, geometry and material.
    \end{itemize}

    \end{column}

    \begin{column}{0.37\linewidth}

    \begin{center}
      \includegraphics[width=0.9\linewidth]{images/fmcw_power_spectrum_spatial.png}
      {\small\itshape Figure 2. Example of RADAR output.}
    \end{center}

    \end{column}

    \end{columns}

    \vspace{2cm}

    So, although RADAR appears to be a cheap and robust sensor for harsh environmental conditions, it is difficult to use without signal processing and/or without combining it with other sensors~\cite{radarsurvey}.

    \end{block}

  \end{column}

	\begin{block}{An application of the radar benefit: The guaranteed dynamic tracking}
      {
        Dynamic objects in the environment need to be identified in order to 

        \vspace{0.5cm}

        \begin{itemize}
          \setlength{\itemsep}{15pt}
          \item Correctly update the map when changes occur.
          \item Avoid dynamic obstacles.
        \end{itemize} 

        \vspace{0.5cm}
        
        Dynamic obstacles are of two types:

        \vspace{0.5cm}

        \begin{itemize}
          \setlength{\itemsep}{15pt}
          \item Semi-static objects does not change its position frequently (e.g., a door).
          \item Highly dynamic objects, where the object moves frequently (e.g., a person).
        \end{itemize}

        \vspace{0.5cm}

        Classically, when range-azimuth sensors are used, dynamic object tracking is performed by comparing multiple frames in order to detect both semi-static and highly dynamic behaviour. When Doppler velocity is available, instantaneous velocity, including angular speed, can be estimated with the Instantaneous Center of Rotation (ICR)~\cite{radartracking}. These kinematic parameters can be estimated using Interval Analysis taking into account bounded RADAR noise.
      }
    \end{block}

	\vspace{0.5cm}
