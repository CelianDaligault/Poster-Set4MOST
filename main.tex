    \begin{block}{Context}
	\textbf{Outdoor unknown off-road navigation stakes in mobile robotics applications are multiples:}

  \vspace{0.5cm}

  \begin{itemize}
    \setlength{\itemsep}{30pt}
    \item 	The need to specify \textbf{ground geometry}, its nature (e.g., slipping ground), but also the presence and nature of \textbf{dynamic objects} to adjust the cartography models and potentially to avoid it when path generation is produced.
    \item The \textbf{change in the environment} (change in luminosity, rain, fog, etc) does not always allow the use of the same sensors continusously. A choice in the sensors but also multisensor fusion, is consequently essential.
    \item The \textbf{noisy nature of acquisition sensors} must be a the subject of careful consideration to bring robust information (possibly guarenteed using Interval Analysis) on which path generation can be based. The context of the described environment corresponds to applications such as construction or farming sites when vehicles evolve jointly with human operators, applications when the security is the top priority.
  \end{itemize}
  \end{block}

    \vspace{0.5cm}

    	\begin{block}{A robust special sensor: The micro-wave RADAR}

      \vspace{0.8em}
      \begin{center}
        \includegraphics[width=1\linewidth]{images/Generalshema}\\[4pt]
        {\small\itshape Figure 1. Overview of the radar integration in SLAM.}
      \end{center}
		
    \vspace{0.5cm}

    Similar to the LiDAR, the RADAR is an electromagnetic-based active sensor, which returns the distance to an obstacle with the time of flight between signal transmission and reception. The difference is that the wave frequencies are in the Radio frequency area, between 3 MHz and 300 GHz (76 - 81 GHz for automotive RADAR), whereas LiDAR wave frequency is in the infrared area, approximately 10,000 times higher. This difference in wave frequency translates into the sensor's property where RADAR:

    \vspace{1.5cm}

    \begin{columns}

    \begin{column}{0.65\linewidth}

    \begin{itemize}
      \setlength{\itemsep}{15pt}
      \item Is less sensitive to signal attenuation in the atmosphere, allowing \textbf{higher maximum range and robustness} to weather conditions such as rain or fog.
      \item Return the \textbf{Doppler velocity} of obstacles.
      \item Can possibly \textbf{go through matter} depending on the type of RADAR
      \item Is \textbf{cheaper} than LiDAR, so attractive for industrial applications
    \end{itemize}

    \vspace{1.5cm}

    However, the RaDAR drawback also include disadvantages compared to the radar:

    \vspace{1.5cm}
    
    \begin{itemize}
      \setlength{\itemsep}{15pt}
      \item The RADAR has a \textbf{poor accuracy} of detections compared to the LiDAR necessitating deeper noise consideration.
      \item It has a \textbf{weak spatial resolution}, which makes it complicated to detect small detection for weak Radar Cross Section target (RCS) depending on shape, geometry and material.
    \end{itemize}

    \end{column}

    \begin{column}{0.35\linewidth}

    \begin{center}
      \includegraphics[width=0.9\linewidth]{images/fmcw_power_spectrum_spatial.png}
      {\small\itshape Figure 2. Exemple of RADAR output.}
    \end{center}

    \end{column}

    \end{columns}

    \vspace{1.5cm}

    So if the RADAR appears to be a cheap and robust sensors for hostile environment conditions it is complicated to use it without signal processing and/or without joining other sensors~\cite{radarsurvey}.

    \end{block}

  \end{column}

	\begin{block}{An application of the radar benefit: The guarenteed dynamic tracking}
      {
        Dynamic objects in environment need to be identified to 

        \vspace{0.5cm}

        \begin{itemize}
          \setlength{\itemsep}{15pt}
          \item Remap correctly the map that has changed.
          \item Avoid dynamic obstacles.
        \end{itemize} 

        \vspace{0.5cm}
        
        The dynamic obstacles are of two types:

        \vspace{0.5cm}

        \begin{itemize}
          \setlength{\itemsep}{15pt}
          \item Semi-static where the object does not change its position often (e.g., a door).
          \item Highly dynamic where the object often move (e.g., a person).
        \end{itemize}

        \vspace{0.5cm}

        Classically, when range-azimuth sensors are used, dynamic object tracking is performed by comparison of multiple frame to be able to detect semi-static and high dynamicity. When Doppler velocity is found, instantaneous velocity, including angular speed, can be found with the Instantaneous Center of Rotation (ICR)~\cite{radartracking}. This kinematics parameters can be found using Interval Analysis taking into accound bounded RADAR noise.
      }
    \end{block}

	\vspace{0.5cm}

     y_r \cos(\theta_i) - x_r \sin(\theta_i) = y_{ICR} \cos(\theta_i) - x_{ICR} \sin(\theta_i) - v_{d_i} \cdot \omega^{-1}
  \end{equation}
